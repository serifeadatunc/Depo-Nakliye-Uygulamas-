# Depo Nakliye Otomasyonu

Python, Tkinter ve Microsoft SQL Server kullanılarak geliştirilmiş depo ve nakliye yönetim sistemidir.

## Özellikler

- Depo stoklarının görüntülenmesi
- Lojistik kayıtlarının takibi
- Satın alma bilgilerinin görüntülenmesi
- Personel bilgilerinin yönetimi
- SQL Server veritabanı entegrasyonu
- Kullanıcı dostu masaüstü arayüz

## Kullanılan Teknolojiler

- Python
- Tkinter
- Microsoft SQL Server
- PyODBC

## Proje Amacı

Depo ve nakliye süreçlerini dijital ortamda yönetmek, stok ve tedarik bilgilerinin hızlı şekilde görüntülenmesini sağlamak amacıyla geliştirilmiştir.
